# 结构(struct)

```v

struct Point {
    x int
    y int 
} 

fn main() {
    p := Point{
        x: 10 
        y: 20 
    } 
    println(p.x) // 使用点访问结构字段
    
    // &前缀返回一个指向struct值的指针。
    // 它被分配到堆上，并自动清除
    // 在函数末尾用v表示，因为它不会被转义。
    pointer := &Point{x:10, y:10} 
    println(pointer.x) // 指针用于访问字段的语法相同 
} 

```
