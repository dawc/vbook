# For 循环

#### V只有一个循环结构：for。

```v

fn main() {
    numbers := [1, 2, 3, 4, 5]
    for num in numbers {
        println(num)
    }
    names := ['Sam', 'Peter']
    for i, name in names {
        println('$i) $name')  // 输出: 0) Sam
    }                         //         1) Peter
}

```

[运行](https://vlang.io/play)

这个for .. in循环用于遍历数组元素。如果需要索引，则可以使用for index, value in来替代。

```v

fn main() {
    mut sum := 0
    mut i := 0
    for i <= 100 {
    sum += i
        i++
    }
    println(sum) // ==> "5050" 
}

```

[运行](https://vlang.io/play)

这种循环形式类似于其它语言中的while循环。

一旦布尔条件求值为false，循环将停止迭代。

同样，条件周围没有圆括号，并且始终需要大括号。

```v

fn main() {
    mut num := 0
    for {
        num++
        if num >= 10 {
            break 
        } 
    }
    println(num) // ==> "10" 
}

```

[运行](https://vlang.io/play)

条件可以省略，这会导致无限循环。

```v

fn main() {
    for i := 0; i < 10; i++ {
        println(i)
    }
}

```
